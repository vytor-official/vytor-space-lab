# 🛰️ Vytor Space Lab

Vytor Space Lab is an early-stage space intelligence software project focused on collecting, analyzing, and visualizing real-time space data using public APIs.

---

# 🚀 Current Version

## V5 — Web Dashboard Development Stage

Current systems:
- Terminal-based dashboard
- NASA APOD integration
- ISS live location tracking
- Asteroid monitoring system
- Flask web dashboard structure

---

# 🌌 Vision

Vytor aims to become a multi-source space intelligence platform capable of:

- Collecting real-time space data
- Analyzing astronomical information
- Visualizing space-related datasets
- Integrating AI-powered analysis systems
- Building accessible space research tools

---

# 📊 Features

## 🛰️ NASA APOD
Displays NASA's Astronomy Picture of the Day.

## 🌍 ISS Tracking
Shows the current live position of the International Space Station.

## ☄️ Asteroid Monitoring
Tracks near-earth asteroid data from NASA APIs.

## 🌐 Web Dashboard
Flask-powered web interface currently under development.
